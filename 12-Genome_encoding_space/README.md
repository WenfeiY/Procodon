## Requirements (Ubuntu 18.04.6 LTS)
- **Python**: >= 3.8
- **pandas**: >= 2.2.3

## Scripts
 - **01-Computing_encoding_space_from_GFF3.py**: Read annotations in GFF3 format, determine available encoding regions, write regions into file and print the rough encoding space (bit).
<br>&emsp;&emsp;Usage: ```python 01-Computing_encoding_space_from_GFF3.py <in_gff3_file> <out_region_file>```

## Attached files
 - **GCA_963668995.1.gff**: Example annotations in GFF3 format, which can be downloaded from [Figshare](https://doi.org/10.6084/m9.figshare.31055530).
